Francais :

La deuxième solution consiste à faire une installation manuelle d'une machine virtuelle prête à accueuillir des fichiers malveillants.

Le point positif d'une installation manuelle est que l'on sait exactement ce que l'on installe et on est beaucoup plus informer.

Le point négatif est que si l'installation n'est pas faite correctement il est possible que certain malware assez bien développer peuvent remonter jusqu'à la machine locale.

---------
Anglais :

The second solution is to do a manual installation of a virtual machine ready to accueuillir malicious files.

The positive manual installation is that you know exactly what is installed and there is much more information.

The downside is that if the installation is not done correctly it is possible that some malware may develop well enough to go back to the local machine.
