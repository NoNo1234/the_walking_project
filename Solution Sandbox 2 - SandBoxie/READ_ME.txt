Francais : 

En installant un logiciel, nous pouvons courir des risques. En effet, il est impossible de contrôler les fichiers accédés ou modifiés par une application. Ainsi, elle peut modifier des fichiers du système à notre insu. Pour y remédier, SandBoxie est une application permettant de sécuriser l’exécution de vos logiciels.

Ainsi en la couplant à une machine virtuelle, il est possible de tester toutes les actions qu'effectuent un éxecutable.

Les caractéristiques principales de Sandboxie : 

Installation de SandBox : SandBoxie vous servira de bouclier contre les actions malveillantes de certains logiciels. En effet, il va installer un Sandbox (bac à sable) entre le disque dur et les applications. Le procédé est simple, à chaque accès disque, il enregistre l’accès dans son SandBox et lorsque le logiciel tente d’écrire sur le disque, il le fait en réalité dans le SandBox, ce qui l’empêche de modifier quoi que ce soit sur votre disque, en toute transparence et sans gêner l’exécution.

Gestionnaire : SandBoxie inclut un gestionnaire qui permet de lister les processus actifs, d’administrer le contenu dans le SandBox et de créer d’autres zones sécurisées. Ces actions peuvent êtres réalisées via l’interface intuitive du logiciel.

Test des logiciels : SandBoxie s’avère extrêmement utile lorsque vous voulez tester des logiciels et ce, sans laisser des traces sur votre machine. Vous pouvez aussi utiliser le logiciel lorsque vous souhaitez effectuer une action dont vous doutez de la sécurité.

Anglais : 

Installing software, we can risk. Indeed, it is impossible to check the files accessed or modified by an application. Thus, it can modify system files without our knowledge. To remedy this, SandBoxie is an application allowing to secure implementation of your software.

And by coupling to a virtual machine, it is possible to test all actions being done by one executable.

The main features of Sandboxie:

Installing SandBox: SandBoxie will be your shield against the malicious actions of some software. Indeed, it will install a Sandbox (Sandbox) between the hard disk and applications. The process is simple, every disk access, it records in its SandBox access and when the software attempts to write to the disk, it is actually in the sandbox, which prevents change anything on your disk, seamlessly and without affecting performance.

Manager: SandBoxie includes a manager who can list the running processes, manage the content in the sandbox and create more secure areas. These shares may be made ​​via the intuitive interface of the software.

Software testing: SandBoxie is extremely useful when you want to test software and without leaving traces on your machine. You can also use the software when you want to perform an action that you doubt the safety.
